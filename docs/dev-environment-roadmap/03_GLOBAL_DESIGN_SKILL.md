# Global Design Skill

Use these rules for every course.

- Theme comes first.
- Strategy comes second.
- Every hazard must affect a realistic golf decision.
- Greens should grow naturally from surrounding terrain.
- Spend rendering budget where the player interacts.
- Reduce detail outside gameplay.
- Adjacent holes are better than empty terrain.
- Avoid isolated floating-hole designs.
- Every course should be recognizable from one screenshot.

Review checklist:
- Is the theme obvious?
- Is the strategy obvious?
- Does every hazard matter?
- Is the environment supporting gameplay?
