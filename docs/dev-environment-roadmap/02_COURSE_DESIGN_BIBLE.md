# Course Design Bible

Every course should have:
1. A Visual Theme
2. A Playing Theme

## Port Links
Visual: Scottish links, fescue, heather, rain, rolling greens.
Playing: Wind, long putts, difficult recoveries.

## Timberline
Visual: Pine forests, mountains, granite.
Playing: Trees define strategy.

## Sable Bay
Visual: Premier coastal golf inspired by Pebble Beach and Pinehurst.
Playing: Ocean wind, carries, precision.

## Red Desert
Visual: Sandstone mesas and canyons.
Playing: Rock hazards, firm bounces, elevated greens.

## Wild Prairie
Visual: Sandhills, native prairie, blowout bunkers.
Playing: Angles, rolling lies, strategic waste areas.
