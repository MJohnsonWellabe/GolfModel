# Marketing clips drop-folder

The marketing kit (`marketing.html`) has five gameplay video tiles. Each tile
shows a still poster until you drop its clip in here. Record the shot on your
phone with the in-game **CLIP** button, then add the file with the matching name:

| Tile            | Filename            |
| --------------- | ------------------- |
| Hole-in-one     | `hole-in-one.mp4`   |
| Check & back up | `backspin.mp4`      |
| Delicate chip   | `chip.mp4`          |
| Clutch putt     | `putt.mp4`          |
| Bunker splash   | `bunker.mp4`        |

MP4 (H.264) plays everywhere; `.mov` from an iPhone also works if you rename the
`<source>` type, but MP4 is safest. Portrait clips fit the tiles best.
